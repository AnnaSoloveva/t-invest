1. список файлов:
	404.php - стр 404
	breadcrumb.php - хлебные крошки
	card-house.php - карточка дома
	contact.php - контакты
	default-text.php - пример обычной тектсовой страницы
	footer.php - подвал
	header-objects.php - шапка у списка объектов. Добавляется менюшка объектов к основной шапке
	header.php - шапка
	index.php - стартовая
	menu.php - главная менюшка
	objects.php - список объектов
